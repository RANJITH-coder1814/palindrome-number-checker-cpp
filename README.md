# 🔄 Palindrome Number Checker (C++)

This project is a simple **C++ program that checks whether a given number is a Palindrome**.

It is a beginner-friendly project that demonstrates **loops, arithmetic operations, and conditional statements**.

---

## 📖 What is a Palindrome Number?

A **Palindrome Number** is a number that remains the same when its digits are reversed.

### Examples

Palindrome Numbers:
121
1331
454

Not Palindrome:
123
456

---

## ⚙️ How the Program Works

1. The user enters a number.
2. The program reverses the digits of the number.
3. The reversed number is compared with the original number.
4. If both are equal, the number is a **Palindrome**.
